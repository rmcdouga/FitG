# Overview
The Detection Routine occurs when a character spaceship is leaving, arriving at, or moving within a planet occupied by an Enemy PDB.

## Trigger Condition
Resolved using the Detection Table (9.7).

## Evasion
Detection depends on the **Evasion Value** of the spaceship/pilot combination.

# Overview
Metal-eating reptile.

## Combat Stats
- **Strength**: 3
- **Endurance**: 3

## Special Rules
- **Target**: Attacks character's *spaceship*, not characters.
- **Damage**: If it is still alive after two rounds of combat, the ship is damaged. If no spaceship is present, no effect.
